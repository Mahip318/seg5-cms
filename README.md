## Project Overview
Tea05 is a specialized POS and Inventory Management System designed for local tea shops. It streamlines the ordering process for baristas while providing real-time inventory tracking and financial reporting for managers.

## Tech Stack
* **Language:** Java 21
* **Database:** MySQL 8.0
* **Middleware:** JDBC (Data Access), custom Auth Service
* **Version Control:** Git/GitHub

## Key Features (Prototype Scope)
1. **Secure Authentication:** Role-based access for Baristas and Managers.
2. **Order Processing:** Interactive menu selection with automated tax calculation.
3. **Inventory Management:** Real-time stock deduction upon sale completion.
